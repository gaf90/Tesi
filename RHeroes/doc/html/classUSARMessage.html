Detailed Description

Message representing data sent to and from USARSim.

The USARMessage class represents a message aimed towards, or signalled by USARController. The class is effectively a hash table containing key-value pairs and an additional string representing the type of the message (e.g. INFO, SEN, STA, DRIVE, ...). When parsing a message the order of the parameters is preserved; similarly when constructing a new message with the provided methods the ordering of the parameters is determined by the order in which the key/value pairs were inserted
