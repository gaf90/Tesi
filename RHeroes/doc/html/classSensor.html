Detailed Description

Generic sensor interface.

The Sensor virtual class defines the common interface that must be implemented by middleware classes that signal sensor data to the main application.

The particular sensorial data emitted (Message) is dependent on the Sensor implementation, hence additional information should be sought in the relative subclass documentation
