Detailed Description

USARSim connection controller.

USARController manages the low-level communication to the USARSim server. A signal is emitted every time a message is received from the server, conversely the controller will forward a message to the server in case a connected Driver has requested so. Both input and output communication data is encapsulated in USARMessage instances
